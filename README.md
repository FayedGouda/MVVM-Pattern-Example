# MVVM-Pattern-Example
In this simple example, I'll show you how to use MVVM pattern in iOS development using Swift, Storyboard and UIKit.
</br>


What we are going to build is a simple list of fake friends where you can:</br></br>
  -Follow a friend</br>
  -Unfollow a friend</br>
  -Visit a friend profile</br>
  <h3>What is MVVM pattern?<h3>
  <h4>Model-View-ViewModel<h4></br>
  Model: will be responsible for data</br>
  ViewModel:Will be responsible for data flow between model and view</br>
  View:Will be responsible for UI elements and user interactions, when it needs data, it will communicate with viewModel to get that data
  also vise versa.</br>
  <h2>NOTE : view here is a (UIViewController+View)<h2></br>
